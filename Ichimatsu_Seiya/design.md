本システムはタスク管理を行うWebアプリである。
ユーザーはタスクの追加・削除・確認ができる。

- 登録されたタスクを一覧表示する
- 各タスクに「完了」「削除」ボタンを表示
- 「追加」ボタンでタスク追加画面へ遷移

- タスク名と期限を入力するフォームを表示
- 「保存」ボタンでタスクを登録
- 登録後は一覧画面へ戻る

- タスクの詳細情報を表示
- 内容の編集が可能

| 項目名 | 内容 |
|-------|------|
| id | タスクの識別子 |
| title | タスク名 |
| deadline | 期限 |
| status | 完了/未完了 |

1. ユーザーがタスクを入力する  
2. 保存ボタンを押す  
3. データを保存する（LocalStorage）  
4. 一覧画面に戻り、タスクを表示する  


- フロントエンド：HTML / CSS / JavaScript  
- データ保存：LocalStorage  
- サーバー：使用しない（ローカルアプリ）
